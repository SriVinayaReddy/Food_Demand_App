# 🍽️ Food Demand Forecasting App

## 📌 Project Description
This project is a web-based application that predicts future food demand using time series forecasting. It helps businesses make better decisions by analyzing past data and forecasting future trends.

---

## 🎯 Problem Statement
Food businesses often struggle with overstocking or understocking due to unpredictable demand. This leads to wastage or loss of sales. This app solves the problem by providing accurate demand predictions.

---

## 🧠 Tech Stack
- Python
- Streamlit
- Prophet
- Pandas
- Matplotlib

---

## ▶️ How It Works
1. Upload your food demand dataset (CSV file)
2. The app processes the data
3. Prophet model is trained on the dataset
4. Future demand is predicted
5. Results are displayed with graphs
